Weather Lookup (gRPC)

Fetches current temperature from DHMZ via a gRPC streaming service running on port 9090. Partial city name matching is supported.

How it works
  1. Browser calls GET /api/weather?city=... (REST bridge).
  2. The REST controller opens a gRPC channel to localhost:9090 and calls WeatherService.GetTemperature.
  3. The gRPC service fetches https://vrijeme.hr/hrvatska_n.xml, parses it, and streams matching <Grad> entries.
  4. Results are returned as JSON and displayed here.
