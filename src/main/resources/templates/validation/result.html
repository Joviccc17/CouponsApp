How it works
  1. When a coupon is saved, XmlGenerationService marshals all coupons to ./data/coupons.xml using JAXB.
  2. This page calls XmlValidationService which creates a JAXB Unmarshaller with a Schema attached.
  3. During unmarshal, JAXB validates each element against coupon.xsd rules (required fields, enum values, types).
  4. Validation events (WARNING / ERROR / FATAL) are collected and displayed here.
